# 散户的A股资产播种与量化复利收割机

面向中国A股和基金自持资产的个人播种计划、持仓成本管理、现金回收与零成本资产沉淀工具，围绕分批配置、资金回收、零成本持有、分红灌溉和纪律复盘进行记录与展示。**独立 App，无后端、无登录、不连接券商、不触达执行，个人数据存本机。**

## 功能模块

| 模块 | 功能 |
|------|------|
| 播种计划 | 输入本金、首批价格/净值、播种批数、下行间距和回收幅度，自动推演股票或基金的分批配置和本金回收方案 |
| 收割计算 | 支持网格和 ATR 两种离线算法，计算播种触发线、回收触发线和归零回收数量 |
| 排雷筛选 | 输入股票代码后自动拉取名称、现价、PE/PB、K线趋势、ST状态，并尽可能拉取质押、财务质量和分红记录 |
| 股票观察 | 添加/删除自选股，实时行情刷新，左滑删除 |
| 股票详情 | 实时报价、近60日走势图、PE/PB估值参考、建仓信号分析 |
| 播种机账本 | 支持股票股数和基金份额，记录每次播种批次、仓位回收、分红/现金派发，自动计算零成本进度 |
| 零成本资产库 | 本金全部回收后自动进入“精神股东荣誉墙”，剩余仓位按纯利润资产展示 |
| 策略纪律 | 将“选种、播种、回本、留种”整理成播种前检查、回收规则、基金适配规则和退出红线 |
| 搜索 | 全市场股票搜索（按代码或名称），一键加入自选 |

## 零成本核心逻辑

```
总投入成本 = 配置价格/净值 × 数量/份额 + 佣金/手续费
已回收资金 = 回收价格/确认净值 × 回收数量/回收份额 + 分红/现金派发
零成本进度 = 已回收资金 / 总投入成本
零成本达成 = 已回收资金 ≥ 总投入成本（剩余持仓/份额等同于零成本）
最新零成本价格 = max(总投入成本 - 已回收资金, 0) / 剩余数量
免费持有数量 = 零成本达成后的剩余股数/份额
```

## 已落地能力

```
播种机账本：记录播种、仓位回收、现金分红/派发，自动汇总已回收本金
零成本资产库：本金全部回收后自动进入荣誉墙；仍有仓位显示纯利润持有，已全部回收显示回本毕业
账本管理：支持无限量股票/基金账本记录、搜索、筛选、排序、单条删除和整标的账本删除
便捷记账：支持截图 OCR 文字粘贴解析，确认后生成播种记录；支持一键生成播种账单并复制导出
策略建议提示：播种计划会计算触发价、回收数量和可保留的零成本数量
网格/ATR收割：根据现价、剩余本金缺口和仓位，计算高抛低吸区间
基础策略池：前5只标的默认纳入播种策略计算提示，超过5只仍可继续做账本管理
自动排雷：输入股票代码后自动填充名称、现价、PE/PB估值分位、ST状态、趋势、触发价、质押、财务和分红数据
硬性过滤：过滤 ST、退市风险、违规担保、财务造假、高质押和现金流为负
红利监控：按股息率、连续分红年限、分红稳定性评估天然灌溉能力
本地提醒：录入当前价、播种线、收割线后在 App 内显示触发提示
```

## 后续可扩展能力

```
深度数据增强：当前已对公开财务/质押/分红接口做多路尝试；若接口缺字段或限流，App 会降级为手工核验
系统推送：接入后台行情轮询后，可从 App 内提醒升级为系统级盘中推送
```

## 播种术功能抽象

```
选种：股票看公司质量，基金看指数/策略、管理人、费率和长期有效性
播种：把计划资金拆成多批，按价格或净值区间执行，不临盘追涨杀跌
回本：达到预设回收条件后记录部分仓位回收，优先回收本金
留种：本金回笼后保留剩余股份或份额，继续跟踪基本面、指数估值和退出条件
```

## 技术栈

- **框架**：Flutter 3.x（iOS + Android 双端）
- **状态管理**：Riverpod 2.x
- **本地数据库**：SQLite（sqflite）
- **数据来源**：核心播种、持仓和策略功能全部本地运行；行情/搜索可使用东方财富公开 HTTP 接口（免费，无需 API Key）
- **图表**：自绘 CustomPainter（K线折线图）

## 运行方式

### 第一步：安装 Flutter

```bash
# macOS 推荐方式（从官网下载，无需魔法）
# 下载地址：https://flutter.dev/docs/get-started/install/macos

# 解压到 ~/development
mkdir -p ~/development
cd ~/development
# 解压下载的 flutter_macos_arm64_xxx.tar.xz

# 添加 flutter 到 PATH（~/.zshrc）
echo 'export PATH="$PATH:$HOME/development/flutter/bin"' >> ~/.zshrc
source ~/.zshrc
```

### 第二步：验证环境

```bash
flutter doctor
```

### 第三步：安装依赖并运行

```bash
cd /path/to/zero-cost-holding-app

# 安装所有依赖
flutter pub get

# 运行到 iOS 模拟器
flutter run

# 或运行到 Android 模拟器
flutter run

# 构建 iOS 包
flutter build ios --release

# 构建 Android APK
flutter build apk --release
```

## 数据说明

播种计划、策略纪律、播种机账本和零成本资产库均为本地计算和本地存储，不需要后端服务。基金支持手工录入代码、名称、净值、份额、现金派发和回收记录；股票搜索和行情页面当前使用公开行情接口。生产化版本建议接入聚合数据、腾讯云金融或其他持牌/合规金融数据服务商，获取A股延迟或准实时行情、历史分红和基金净值数据。
